# EQ（EI）  
Emotional Intelligence Quotientの略。SIY本の原語ではEI（Emotional Intellijence）。  
こころの知能指数、情動知能と訳される。  
自分自身と他人の気持ちや情動をモニターし、見分け、その情報を使って自分の思考や行動を導く能力。  
EQは5つの領域に分類される。  
```  
自己認識... 自分の内面の状態、好み、資質、直感を知ること  
自己統制... 自分の内面の状態、衝動、資質を管理すること  
モチベーショニング... 目標達成をもたらしたり助けたりする情動的な傾向  
共感... 他人の気持ち、欲求、関心を認識すること  
社会的技能... 他人から望ましい反応を引き出すのに熟達していること  
```  
自己認識、自己統制、モチベーショニングは内省的知能(イントラパーソナル)、  
共感、社会的技能は対人的知能(インターパーソナル)に分類される。  
EQの向上により、3つの恩恵が得られる。  
```  
優れた職務遂行能力  
抜群のリーダーシップ  
幸せをお膳立てする能力  
```  
  
# マインドフルネス  
特別な形、つまり意図的に、今の瞬間に、評価や判断とは無縁の形で注意を払うこと。  
  
# SIY  
マインドフルネスに基づくEQ向上カリキュラムの名前。サーチインサイドユアセルフの略。  
注意力、自己認識、自己統制、モチベーショニング、共感、社会的技能のトレーニングステップから成る。  

## 注意力のトレーニング  
EQ向上トレーニングに必要な注意力を養うトレーニング。  

### 【エクササイズ】マインドフルネス瞑想
注意力の基本能力を養うエクササイズ。  

### 【エクササイズ】歩く瞑想
活動中にマインドフルネスな状態になる能力を養うエクササイズ。  

### 【エクササイズ】マインドフル・リスニング
他人に向けてマインドフルネスな状態になる能力を養うエクササイズ。  

### 【エクササイズ】喜びのマインドフルネス瞑想
集中を手軽にし、瞑想中のスッカを加速させ、スッカを日常生活に浸透させることができるエクササイズ。  

### 【エクササイズ】集中と開放のサーキットマインドフルネス瞑想
集中と開放の両方のメタ注意を強くするエクササイズ。両方の注意の明瞭さと鮮明さが強くなる。  
努力とリラクゼーションのバランスが重要視され、鍛錬すれば安らぎとフローの状態にはいれる。  
  
メタ注意：注意力が逸れていることに対する注意  
フロー：心が穏やかで集中した状態  
  
## 自己認識のトレーニング  
EQの自己認識領域を向上させるトレーニング。  
自己認識は3つの領域で構成される。  
```  
情動の自覚... 自分の情動とその影響に気づくこと  
正確な自己査定... 自分の長所と限界を知ること  
自信... 自分の価値と能力を強く実感すること  
```  
情動の強い自覚が、より正確な自己査定につながり、それが前より強い自信を生む。  

### 【エクササイズ】ボディスキャン
情動の自覚能力(生理的経験である情動を高解像度で自覚する力)を養うエクササイズ。  

### 【エクササイズ】ジャーナリング
自己査定能力と自信を養うエクササイズ。  
  
## 自己統制のトレーニング  
EQの自己統制領域を向上させるトレーニング。  
自己統制は5つの領域で構成される。  
```  
自制心... 破壊的な情動や衝動を抑える  
信頼性... 正直さと誠実さの基準を維持する  
良心性... 自分の振る舞いに責任を取る  
適応性... 変化に柔軟に対応する  
革新性... 新奇な考え方やアプローチや新しい情報を気兼ねなく受け入れる  
```  
本来5つとも選択できるはずが、情動による衝動のせいで不可能になっている。  
自己統制は情動を避けたり抑え込んだりすることではなく、上手くあしらうことで選択を可能にする。  
  
### 【思考パターン】執着と嫌悪の二つを捨てる
苦しみ(嫌悪)抜きで痛みを、後で不満(執着)を覚えずに喜びを経験する原理。    
私たちが何かの現象を経験するときは、感覚器官と対象が接触し、知覚した直後に執着か嫌悪が湧き起こる。  
マインドフルネスで知覚と執着/嫌悪を区別する。  
  
### 【思考パターン】痛みがないときを知る
苦悩に対処するための一般原理。  
痛みがないことを意識し、苦悩の不在に絶えず気づくことで、痛みからの解放の心地よさを楽しむように促され、幸せになれる。

### 【思考パターン】いやになることにいやにならない
苦悩に対処するための一般原理。  
人間は苦悩を抱くことに対して嫌悪感を持ってしまう。  
そうではなく苦悩は自然に湧き起こるものだと理解し、決してそれに対して嫌にならない。  

### 【思考パターン】怪物どもに餌をやらない
苦悩に対処するための一般原理。  
やってきた不安や怒りの怪物に対して、自分の心がそれを煽る話を繰り返し語っていることに気づいたらやめること。すると怪物は去っていく。  

### 【思考パターン】あらゆる思考を優しさとユーモアを持って始める
苦悩に対処するための一般原理。  
自分のしくじりをユーモアにとらえ、あらゆる思考を自分と他人への優しさと思いやりを持って始めること。  

### 【エクササイズ】シベリア北鉄道
誘発に対処するエクササイズ。  
誘発(トリガー)：些細な事柄が不釣り合いなまでに大きな情動反応をこと。トリガーされたとき3点に注意を払う。
```
肉体... 呼吸が浅くなる、鼓動が速まる、むかむかする。
情動... 闘争・逃走反応を経験し、情動の爆発を起こすか、「自動車のヘッドライトを浴びたシカ」のような気持ちになる。
思考... 被害者のような気持がする、非難や批判が頭に浮かぶ、注意を払うのが困難になる。
```

## モチベーショニングのトレーニング  
EQのモニベーションニング領域を向上させるトレーニング。  
仕事のコンテキストにおける幸せは3つから生まれる。  
```  
快楽... ロックスターのような快楽  
情熱... 「フロー」とも呼ばれる。物事に没頭し最高の仕事(ピークパフォーマンス)を見せ、時間が瞬く間に過ぎていく。  
崇高な目標... 自分より大きくて、自分にとって意味のあることの一部になる。  
```  
快楽から生まれる幸せは、持続性が低く、快楽を与えてくれる刺激がなくなったり習慣化すると基本設定に戻る。  
情熱(フロー)から生まれる幸せは快楽のものより持続性があり、習慣化もしない。  
そして崇高な目標から生まれる幸せは最も持続性が高い。利他的な目標であればなおさら。  
  
モチベーショニングには3つのステップがある。  
```  
整合性... 自分の仕事を自分の価値観や崇高な目標と整合させる  
想像... 自分にとって望ましい未来を思い描く  
回復力... 自分の行く手に待ち受ける障害を克服する能力  
```  

### 【エクササイズ】価値観と崇高な目標を発見する
整合性のエクササイズ。  
自分の価値観や崇高な目標を自分自身にはっきりさせる。  
そして内発的モチベーションに従って仕事につく。  
  
内発的モチベーション : 優れた仕事ぶりを引き出すのに最善なのは金銭などの外的な報酬ではなく、  
自律、熟達、目的から成る「内発的モチベーション」と呼ぶ自分の中に見つかる動機である。  
```  
自律... 自分の人生を自分で導きたいという衝動  
熟達... 大切なことがもっともっと上手にできるようになりたいという要望  
目的... 自分のすることを、自分自身よりも大きいもののためにしたいという強い思い  
```  
  
### 【エクササイズ】自分の理想の未来を見つける
想像のエクササイズ。  
頭の中で理想の未来を想像し、発見し、強化する。  
  
### 【エクササイズ】回復力の瞑想
回復力のエクササイズ。  
整合性と想像は自分の行きたい場所を見つける助けになり、回復力はそこに行く着くのを助ける。  
回復力は3つのレベルで鍛えられる。  
```
内面の穏やかさ... 心の中で内面の穏やかさに一貫してアクセスできるようになれば、その上にあらゆる楽観と回復力を築ける。
情動的な回復力... 成功も失敗も情動的な経験であり、それらに対処する能力を伸ばせる。
認知的な回復力... 挫折を自分にどう説明するかを理解し、有効な心の習慣を生み出せば、楽観を育む助けとなる。
```

## 共感のトレーニング
EQの共感領域を向上させるトレーニング。  
私たちは他人の生理的状態を真似ることで共感する。この同調には自己認識と同じ装置が使われる。  
そして共感は信頼を築く。信頼は有能なチームを作るには不可欠の土台である。  
  
共感は優しさとともに深まる。相手に優しくすればするほと、相手とうまく共感できる。  
また相手との類似を感じることでも共感は深まる。  
  
私たちは何かを考えると、そのような人間になる。
つまりエクササイズによって類似点を見出して優しさを差し伸べる人間となれる。

### 【エクササイズ】私と全く同じ
他の人々がどれほど自分に似ているか思い起こし、平等という心の習慣を生み出すエクササイズ。  

### 【エクササイズ】愛情に満ちた優しさ  
他人の幸せを願い、優しさという心の習慣を生み出すエクササイズ。

### 【思考パターン】誠実さ、優しさ、率直さから始める
信頼を築くための考え方。  
相手は善良な人で、そうでないことがわかるまでは善良な人として扱われるに値すると思って関わること。  
そのように認知するだけではなく情動脳も説得し、それから優しさと率直さを持って語り合うこと。  

### 【思考パターン】3つの仮定
チームを相互信頼へと向かわせる慣行。  
チーム全員で3つの仮定を共有する。  
```
1. この部屋の人は全員、個人の利益を超えるもののためにここにいる。
2. なので誰ひとり胸に秘めた狙いなど持っていない。
3. なのでたとえ意見が合わないときにさえ分別ある行動がとれる。
```

### 【エクササイズ】共感的なリスニング
相手から最善のものを引き出す能力を向上させるエクササイズ。

### 【思考パターン】上手に人を褒める
相手から最善のものを引き出す1つの方法。  
頭の良さを褒める「個人称賛」ではなく、取り組みを褒める「プロセス称賛」を行うこと。  
  
個人称賛されると、人間の成功はもって生まれた不変の才能のおかげだという「固定的な姿勢」が強まり、  
失敗すると個人的な欠点のせいにし、失敗したら不適切な人間であることが判明しかねないときには危険を冒すことを恐れるようになる。  
  
プロセス称賛されると、人間の才能は熱意や努力によって伸ばすことができ、成功はそれらから導かれるという「成長志向の姿勢」が強まり、  
大きな業績を上げるのに欠かせない向上心や回復力が生まれる。  
  
### 【エクササイズ】政治的意識のエクササイズ
組織の情動の流れと力関係を読む能力を向上させるエクササイズ。

## 社会的技能のトレーニング
EQの社会的技能領域を向上させるトレーニング。  
社会的技能は3つの領域で構成される。  
```
思いやりのあるリーダーシップ
善意に基づく影響力の行使
洞察力に溢れたコミュニケーション
```
思いやりとは、他者の苦しみに対する気遣いの感覚と、その苦しみが取り除かれるようにしたいという強い願望とを伴う心の状態である。  
3つの要素からなる。  
```
認知的な要素... 「私はあなたを理解している」
情動的な要素... 「私はあなたに同情する」
動機づけの要素... 「私はあなたの役に立ちたい」
```
また、思いやりとは「私」から「私たち」への移行でもある。  

### 【エクササイズ】善良さを増す瞑想
善良さを伸ばすことで思いやりの力を鍛えるエクササイズ。

### 【エクササイズ】トングレンの瞑想  
上級者向けの、善良さを伸ばすことで思いやりの力を鍛えるエクササイズ。

### 【思考パターン】SCARF(スカーフ)モデル
影響力を効果的に行使するための考え方。
脳は、基本的に報酬には接近し、脅威は回避する機械である。脅威の回避は報酬への接近よりも3倍強く反応する。  
脳がおもな報酬あるいは脅威として扱う社会的経験を五つの領域に分けた「SCARF(スカーフ)モデル」というものがある。  
「Status(地位)」「Certainty(確実性)」「Autonomy(自律性)」「Relatedness(関係性)」「Fairness(公平性)」  
自分の行動が自分と他者のためにSCARF要因をどう強化できるか理解することで、自分の影響力を広げることができる。
  
### 【思考パターン】洞察力に溢れたコミュニケーション。
厄介な会話を行うための枠組み。  
共感能力が高くても、洞察力が欠けているとイライラする会話に巻き込まれる。  
うまくこなすためには5つのステップがある。  
```
1. 「三つの会話」を予行演習することで準備する
2. 話題を切り出すかどうか決める
3. 客観的な「第三者の話」から始める
4. 相手の話と自分の話を吟味する
5. 問題を解決する
```
##### 1. 「三つの会話」を予行演習することで準備する
どんな会話でも実際には三つの会話が行われている。  
```
内容に関する会話... 何があったか？
気持ちに関する会話... どんな情動がかかわっているか？
アイデンティティに関する会話... ここから自分について何がわかるか？
```
アイデンティティに関する会話では、  
・私は有能か？  
・私は善良か人間か？  
・私は愛される価値があるか？  
の3つの疑問のうちどれか一つが、必ずと言っていいほど入っている。  
  
三つの会話それぞれを整理しておく。  

##### 2. 話題を切り出すかどうか決める
話題を切り出すことで、何を成し遂げようと望んでいるのか。  
問題を解決したり誰かが能力を伸ばすのを助けたりするような建設的な意図なのか、  
それともただ誰かに嫌な思いをさせる非建設的な意図なのかを考え、話題を切り出す必要があるのか考える。  

##### 3. 客観的な「第三者の話」から始める
「第三者の話」とは、状況全体を把握している公平無私な第三者の立場から見たときに、物事がどう起こったかを説明したものだ。  
最も客観的な第三者の話から始めると、相手との共通の基盤を築ける。

##### 4. 相手の話と自分の話を吟味する
相手の話に耳を傾け、共感し、自分の話を伝える。そのとき同じ状況の解釈がどう違っているのか吟味し、  
それぞれの情動の原因を知るための会話をする。  

##### 5. 問題を解決する
お互いの利益を守り続けられるように、双方にとって最も重要な関心や利益に応える解決策を考える。  

### 【エクササイズ】厄介な会話に備える
洞察力を持って厄介な会話をこなすことに備えるエクササイズ。

### 【エクササイズ】マインドフルなメール送信
メールによる効果的なコミュニケーションに大切な洞察を得るエクササイズ。
